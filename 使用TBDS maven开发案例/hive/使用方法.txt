运行方法一：
1、首先设置HADOOP_CLASSPATH环境变量：
export HADOOP_CLASSPATH=$HADOOP_CLASSPATH:/usr/hdp/2.2.0.0-2041/hive/lib/*:/etc/hive/conf;

2、运行程序
hadoop jar hivejdbc-1.0-SNAPSHOT.jar com.hivetest.hivejdbc.HiveJdbcTest 172.16.32.13 testdb testuser testuserpasswd sqlstring
其中5个参数含义：
172.16.32.13        表示hiveserver2 的ip
testdb              表示hive数据库名字
testuser            表示用户名（登录portal门户页面的用户）
testuserpasswd      表示用户密码（登录portal门户页面的用户密码）
sqlstring           表示sql查询语句。由于sql查询语句是由多个string字符串构成的，为了作为一个args参数传递为java程序，需要用""双引号括起来，当做一个参数来使用。



运行方法二：
java -Djava.ext.dirs=/usr/hdp/2.2.0.0-2041/hive/lib:/usr/hdp/2.2.0.0-2041/hadoop -classpath /etc/hive/conf:./hivejdbc-1.0-SNAPSHOT.jar com.hivetest.hivejdbc.HiveJdbcTest 172.16.32.13 testdb testuser testuserpasswd sqlstring



运行方法三：
java -classpath ./hivejdbc-1.0-SNAPSHOT.jar:/usr/hdp/2.2.0.0-2041/hive/lib/*:/usr/hdp/2.2.0.0-2041/hadoop/client/*:/etc/hive/conf com.hivetest.hivejdbc.HiveJdbcTest 172.16.32.13 testdb testuser testuserpasswd sqlstring
